# Vehicle Detection in Real-Time Environment During Foggy Conditions

## 📌 Project Overview
This project presents a real-time vehicle detection system for traffic monitoring in foggy weather conditions using MATLAB. The system uses computer vision and image processing techniques to enhance low-visibility images and accurately detect vehicles.

## 🎯 Objectives
- Detect vehicles in real-time under foggy conditions.
- Improve image visibility using fog reduction techniques.
- Enhance detection accuracy through feature extraction methods.

## 🛠️ Technologies Used
- MATLAB
- Computer Vision
- Image Processing
- Feature Extraction Techniques

## ✨ Features
- Real-time vehicle detection
- Fog reduction and image enhancement
- Vehicle identification and tracking
- Performance evaluation under different fog conditions

## 📂 Project Files
- MATLAB source code
- Project report
- Sample input and output images
- Results and screenshots

## 🚀 Applications
- Intelligent Transportation Systems (ITS)
- Traffic surveillance and monitoring
- Road safety in adverse weather conditions
- Smart city transportation solutions

## 📈 Future Scope
- Integration with deep learning models for higher accuracy.
- Deployment on embedded systems for real-time implementation.
- Extension to detect multiple vehicle types and traffic violations.
